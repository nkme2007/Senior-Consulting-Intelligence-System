# Skill Chains — Complete Library

> The real power of the Senior Consulting Intelligence System is how skills compound. This is the complete chain library — 15+ sequences for every consulting scenario.

---

## How to Read These Chains

```
SKILL_A → SKILL_B → SKILL_C
Step 1   → Step 2   → Step 3
```

**Arrow (→)** = explicit handoff. When a skill completes, it states what's running next and why.

**Plus (+)** = skills running simultaneously or in the same session.

**TM** can be inserted at any point as a modifier — e.g., `CIM → DIE | Mode: McKinsey`

## The Core Chains

---

### 1. Full Initiative Chain
*New project or strategy from scratch*

```
PEA → DIE → MKA → EBE → SAS → ENB
```
| Step | Skill | What happens |
|---|---|---|
| 1 | PEA | Sharpen the brief |
| 2 | DIE | Structure the decision |
| 3 | MKA | Define what success looks like |
| 4 | EBE | Build the execution plan |
| 5 | SAS | Simulate stakeholder resistance |
| 6 | ENB | Build the leadership narrative |

**Best for:** New programmes, transformations, major investments

---

### 2. Problem First Chain

|---|---|
| Starting something new | Full Initiative |
| Stuck on a problem | Problem First |
| Need exec approval | Leadership Pitch |
| Need budget | Budget Approval |
| New market or threat | Market Entry / Competitive |
| Plan is stalled | Execution Recovery |
| Setting OKRs | Metrics-First |
| Under competitive threat | Competitive Defence |
| Inherited a system | Institutional Foundation |
| Major transformation | Org Transformation |
| Post-acquisition | M&A Integration |
| Regulatory / audit | Governance / Audit |
| New in a role | New Leader Onboarding |
| Quick single decision | Quick Decision |
