# PTA — Problem-to-Action
**Code:** `PTA` | **Position:** Skill 6

> *From root cause to sprint ticket — no gap.*

---

## What This Skill Does

A three-module skill that takes a problem from raw diagnosis all the way to actionable tasks — with no gap between thinking and doing.

---

## Three Modules

**Module 1 — Problem Deconstructor**
MECE root cause analysis: what's actually broken, and why.

**Module 2 — Framework Builder**
Selects and builds the right decision framework for the problem type: weighted scoring, decision tree, 2x2, or custom.

**Module 3 — Action Converter**
Converts a decision into sprint or waterfall tasks — with owners, effort estimates, and sequencing.

---

## When to Use

- You need to diagnose a problem deeply before solving it
- You need the right framework — not just any framework
- You have a decision and need it converted into tasks immediately

**Trigger phrases:** *"Root cause / MECE / which framework / sprint tasks / task plan"*

---

## Output

- **Core problem statement** — precise, one sentence
- **MECE cause tree** — exhaustive, non-overlapping root cause map
- **Signal vs. noise filter** — separating symptoms from causes
- **Framework selection + build** — the right tool for this specific problem
- **Stakeholder narrative** — how to explain the diagnosis to others
- **Sprint/waterfall task plan** — execution-ready tasks with owners
